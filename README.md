# Two-tone Rotating Circle

A simple webpage with a centered, two-tone spinning ball. Use the slider to control rotation speed in real-time. (Made with AI.)

## How to view

- Open `index.html` in your browser (double-click or drag into browser).

## Features

- **Centered Circle**: Two-tone gradient (red/pink and blue) that rotates smoothly.
- **Speed Slider**: Adjust rotation speed from 0.5s to 5s per rotation.
- **Responsive**: Works on all screen sizes.
- **Accessible**: Respects `prefers-reduced-motion` preference.

## Customize

Edit `styles.css` to change:
- `--c1` and `--c2`: gradient colors
- `--size`: circle diameter
- `--bg`: background color

## Files

- `index.html`: Page structure with slider controls
- `styles.css`: Styling and animation
- `script.js`: Slider interaction logic
